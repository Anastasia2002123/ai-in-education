Завдання 3. Побудуй просту модель класифікації

Для цього завдання рекомендується використовувати Google Colab та бібліотеку scikit-learn.

Ось приклад коду, який можна скопіювати і запустити в Google Colab:

from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.neighbors import KNeighborsClassifier
from sklearn.metrics import accuracy_score

# Завантаження даних
iris = load_iris()
X = iris.data
y = iris.target

# Розділення на тренувальні та тестові
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

# Створення моделі
model = KNeighborsClassifier(n_neighbors=3)
model.fit(X_train, y_train)

# Прогнозування
y_pred = model.predict(X_test)

# Оцінка точності
accuracy = accuracy_score(y_test, y_pred)
print(f"Точність моделі: {accuracy:.2f}")

Можеш змінювати параметри n_neighbors і дивитися, як це впливає на точність.

Завдання 4. Аналіз фейкових новин

Завантаж набір даних: Fake News Dataset на Kaggle

Кроки для виконання завдання у Google Colab або Jupyter Notebook:

  1. Очисть текст новин: видали HTML-теги, стоп-слова.
  2. Побудуй модель логістичної регресії або дерева рішень для класифікації.
  3. Використай CountVectorizer або TfidfVectorizer для векторизації текстів.
  4. Проаналізуй слова, які найчастіше зустрічаються у фейкових новинах.

Ось приклад коду для Google Colab (скорочений):

import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.linear_model import LogisticRegression
from sklearn.metrics import classification_report

# Завантаження даних (потрібно спочатку завантажити CSV у Colab)
df = pd.read_csv('Fake.csv')

# Очистка тексту - можна додати власні функції

# Розділення
X_train, X_test, y_train, y_test = train_test_split(df['text'], df['label'], test_size=0.3, random_state=42)

# Векторизація
vectorizer = TfidfVectorizer(stop_words='english', max_features=10000)
X_train_tfidf = vectorizer.fit_transform(X_train)
X_test_tfidf = vectorizer.transform(X_test)

# Модель
model = LogisticRegression(max_iter=1000)
model.fit(X_train_tfidf, y_train)

# Прогнозування
y_pred = model.predict(X_test_tfidf)
print(classification_report(y_test, y_pred))

# Аналіз найважливіших слів
feature_names = vectorizer.get_feature_names_out()
coef = model.coef_[0]
top_fake_words = [feature_names[i] for i in coef.argsort()[-20:]]
print("Слова, що найбільше асоціюються з фейковими новинами:", top_fake_words)
